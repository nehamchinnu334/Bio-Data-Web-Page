📄 Bio Data Web Page

A simple and structured Bio Data Web Page developed using HTML.
This project demonstrates the use of fundamental HTML elements to create a personal profile webpage with organized content and proper formatting.

🚀 Project Overview

The Bio Data Web Page displays personal information in a clear and structured format. It includes sections such as:

*Name and Contact Information
*Career Objective
*Educational Qualification
*Hobbies
*Skills
*External and Internal Links

The project focuses on understanding HTML structure, formatting tags, tables, lists, and basic styling using inline CSS.

🛠️ Technologies Used
*HTML5
*Inline CSS

📚 Concepts Implemented

*Semantic HTML structure
*Headings (h1, h2)
*Paragraph formatting
*Text styling (b, i)
*Image alignment

Tables with:
*rowspan
*thead and tbody
*Ordered and Unordered Lists
*Hyperlinks
*Inline styling

📂 Features

✔ Clean and structured layout
✔ Educational qualification table with proper formatting
✔ Organized hobbies and skills sections
✔ External website linking
✔ Simple and beginner-friendly design

🎯 Learning Outcome

Through this project, I gained practical experience in:
*Structuring web pages using HTML
*Designing tables and lists effectively
*Using inline CSS for basic styling
*Organizing content in a readable format

🔗 How to Run

*Download or clone the repository
*Open index.html in any web browser
*View the Bio Data webpage
